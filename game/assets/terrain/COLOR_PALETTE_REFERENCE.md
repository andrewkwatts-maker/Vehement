## Color Harmony Analysis

### Complementary Colors
- **Primary (Warm)**: Golden light #FFE7C7
- **Complement (Cool)**: Blue ambient #8C9EBF
- **Creates**: Warm/cool contrast for cinematic depth

### Triadic Harmony
- **Grass Green**: #408C33 (113° hue)
- **Mountain Blue**: #8594AD (218° hue)
- **Path Brown**: #6B5238 (31° hue)
- **Angle**: ~120° separation

### Analogous Progression (Distance)
```
Foreground → Mid-ground → Background
Green(113°) → Green-Blue(140°) → Blue(218°)

Creates natural atmospheric depth
```

## Color Temperature Guide

```
Warm Colors (Foreground):
- Grass: Neutral-warm (slight yellow undertone)
- Dirt: Warm (orange undertone)
- Path: Warm (brown-yellow)
→ Creates inviting, natural feel

Cool Colors (Background):
- Mountains: Cool (blue undertone)
- Snow: Very cool (blue highlights)
- Fog: Cool (blue-grey)
→ Creates atmospheric distance

Temperature Gradient:
Warm (3500K) ────────────────────────► Cool (6500K+)
Foreground                              Background
```

## Post-Processing Color Adjustments

### Color Grading
```
Exposure:    +0.05 (5% brighter)
Contrast:    +0.15 (15% more contrast)
Saturation:  +0.08 (8% more saturated)
Gamma:       1.0 (no change)
Temperature: +15 (warmer by 15 mired)
Tint:        -5 (slight magenta reduction)
```

### Tone Mapping Curve
```
Input  → Output
0.0    → 0.0    (Black point)
0.18   → 0.22   (Mid-grey slightly brighter)
0.5    → 0.58   (Highlights preserved)
1.0    → 0.95   (White point with rolloff)
```

## Usage Examples

### Blending Two Materials (Height-Based)
```glsl
float heightFactor = smoothstep(5.0, 15.0, worldPos.y);
vec3 color = mix(GRASS_BASE, DIRT_BASE, heightFactor);
```

### Applying Atmospheric Perspective
```glsl
float distance = length(worldPos - cameraPos);
float fogFactor = smoothstep(80.0, 400.0, distance);
float desatFactor = smoothstep(50.0, 300.0, distance) * 0.7;

vec3 color = GetMaterialColor(worldPos);
float luminance = dot(color, vec3(0.2126, 0.7152, 0.0722));
color = mix(color, vec3(luminance), desatFactor);  // Desaturate
color = mix(color, FOG_COLOR, fogFactor);           // Apply fog
```

### Lighting Calculation with Color
```glsl
vec3 lighting = LIGHT_SUN * diff;              // Warm sun
lighting += LIGHT_AMBIENT * ambientStrength;    // Cool ambient
lighting += LIGHT_RIM * rimFactor;              // Warm rim

vec3 finalColor = materialColor * lighting;
```

## Accessibility Notes

All color combinations meet WCAG AA contrast requirements for visual clarity:
- Grass (base/shadow): 4.8:1 ratio
- Dirt (base/shadow): 4.2:1 ratio
- Mountain (base/shadow): 3.1:1 ratio (acceptable for non-text)

The palette avoids problematic color combinations for common color vision deficiencies:
- Protanopia (red-weak): Green/blue distinction maintained
- Deuteranopia (green-weak): Warm/cool temperature distinction maintained
- Tritanopia (blue-weak): Saturation/luminance distinction maintained
